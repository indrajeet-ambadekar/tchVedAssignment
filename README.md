## Build & development

Run `npm install`
Run `bower install`
Run `npm install --save-dev gulp gulp-cli`
Run `gulp serve`

### On running the last command, a local node server will fire up on port 9000 and you can access the app from the url 

`http://localhost:9000/#!/`
